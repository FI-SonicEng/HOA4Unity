Detailed Description

The Maths class owns a set of useful static methods.

The Maths class owns a set of useful static methods to clip and wrap angles and to convert coordinates from cartesian to spherical or from spherical to cartesian with the \(\frac{\pi}{2}\) offset special feature.

The abscissa converter function.

This function takes the radius \(\rho\), the azimuth \(\theta\) and the elevation \(\varphi\) of a point and retrieves the abscissa \(x\).

\[x = \rho \cdot cos{\left(\theta + \frac{\pi}{2}\right)} \cdot cos{(\varphi)} \]
